# customer-migration
Проект включает в себя:

- **Фронтенд часть:** Vue3  
- **Бекенд часть:** ASP.NET Core  
- **База данных:** MSSQL  

---

```bash
Актуальная ветка - develop

Для запуска фронтенд части необходимо:
1. Установить зависимости в корневой директории фронтенда, выполнить команду  npm install
2. Запустить дев-сервер в корневой директории фронтенда, выполнить команду npm run dev

Для поднятия базы данных необходимо:
1. В корневой директории проекта бекенд части выполнить команду docker-compose up -d

Для запуска бекенд части необходимо:
1. В корневой директории бекенда выполнить команду dotnet run

Работа с данными:
Папка data содержит в себе .xlsx файл со всеми данными клиентов
XML файлы генерируются в папке ./XmlImports
XML файлы архивируются в папку ./XmlImports/Archive
```

# Содержание проекта:
- **Содержит логику миграции данных из Excel файлов с расширением .xlsx в базу данных MSSQL**
- **Содержит логику фоновой генерации XML файлов, их парсинга и последующего сохранения в базу данных**
- **Содержит логику обновления сущностей в базе данных и их пагинированного вывода**
